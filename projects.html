FreeTime:

This app was created using React Native for my CS262 Software Engineering class at Calvin University.


It allows you and your friends/classmates/family/etc. to create a group together and respectively input the times you are free to hang out during the week. The app will then calculate the time periods of full overlap among group members.

Trivia:

This web app was created using Angular for my CS336 Web Development class at Calvin University


Create a free account, choose a difficulty and category for trivia questions, and see how well you can do. The site pulls questions from opentdb.com and keeps track of your lifetime account statistics with Cloud Firestore.
